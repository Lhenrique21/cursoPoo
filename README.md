Exercícios de Programação Orientada a Objetos (Java)
Este repositório contém uma coleção de exercícios práticos realizados durante o curso de Java e POO. O objetivo principal é consolidar o domínio dos pilares da Orientação a Objetos e a sintaxe da linguagem Java.

🎯 Objetivos de Aprendizado
Através destes exercícios, foram praticados conceitos fundamentais como:

Classes e Objetos: Estruturação de entidades do mundo real.

Encapsulamento: Proteção de dados com modificadores de acesso e métodos assessores.

Herança e Polimorfismo: Reutilização de código e comportamentos dinâmicos.

Abstração: Criação de classes e métodos abstratos.

Interfaces: Definição de contratos entre classes.
